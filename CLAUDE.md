# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## What this repo is

A personal recipe collection of ~286 Markdown files with YAML frontmatter. Four index files are auto-generated by a GitHub Action whenever recipes change — do not edit the index files manually.

## Index generation

```bash
python generate_indexes.py
```

Regenerates all four index files (`Recipe-Index-flat.md`, `Recipe-Index-canonical.md`, `Recipe-Index-category.md`, `Recipe-Index-health.md`). This runs automatically on push via `.github/workflows/generate-indexes.yml`. Requires PyYAML (`pip install pyyaml`).

## Recipe frontmatter schema

Every recipe file in `recipes/` must have these six frontmatter fields:

```yaml
title: Human-readable recipe name
slug: lowercase-hyphen-separated
cuisine: <see canonical list below>
category: <see canonical list below>
health_rating: <0–5 integer>
health_rating_label: <matching label>
```

## Canonical cuisine list

African, Asian, Balkan, British, **Cantonese**, Caribbean, Chinese, Filipino, French, Indian, Italian, Japanese, Mediterranean, Mexican, Middle Eastern, North American, Persian, Southern / Cajun, Spanish, Thai, Vietnamese

> **Important:** If a recipe being added uses a cuisine not in this list, flag it to the user before writing the file. Do not silently pick the closest match — ask whether to use an existing cuisine or add a new one.

## Canonical category list

Appetizers, Breads & Pastries, Breakfast & Brunch, Dessert, Dressings & Marinades, Drinks, Mains, Pasta, Salads & Sides, Sauces & Condiments, Snacks, Soups & Stocks, Spice Mixes & Pastes

> **Important:** If a recipe being added belongs to a category not in this list, flag it to the user before writing the file.

## Health rating scale

| Rating | Label                     |
|--------|---------------------------|
| 5      | Very Healthy Everyday     |
| 4      | Generally Healthy         |
| 3      | Mixed / Context-Dependent |
| 2      | Rich / Heavy              |
| 1      | Treat / Indulgent         |
| 0      | Unrated                   |

The integer and label must match the same row exactly.

## Cuisine notes

- **Cantonese** covers Cantonese, Hong Kong, and Macau dishes — do not use Chinese for these.
- **Asian** is a fallback for pan-Asian dishes that don't fit a specific cuisine.
